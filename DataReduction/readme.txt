Data Reduction

----------------------------------------------------------------
                            PCA.m
----------------------------------------------------------------
Performs principal component analysis on a dataset and plots 1st and 2nd
principal components


HOW TO RUN:

To run from the Matlab command line use:

	PCA()

The default file used will be 'diabetes.csv'.
To use a different file use:

	PCA(file name)



----------------------------------------------------------------
			    LDA.m
----------------------------------------------------------------
Reduces a dataset to one dimension using linear discriminant analysis


HOW TO RUN:

To run from the Matlab command line use:
	
	LDA()

The default file used will be 'diabetes.csv'.
To use a different file use:

	LDA(file name)



----------------------------------------------------------------
			IG.m
----------------------------------------------------------------
Ranks features by information gain


HOW TO RUN:

To run from the Matlab command line use:
	
	IG()

The default file used will be 'diabetes.csv'.
To use a different file use:

	IG(file name)



----------------------------------------------------------------
		kMeansCluster.m
----------------------------------------------------------------
Performs k-means clustering using just the 6th and 7th features of the data


HOW TO RUN:

To run from the Matlab command line use:
	
	kMeansCluster()

The default file used will be 'diabetes.csv'.
To use a different file use:

	kMeansCluster(file name)



Note: The full file path should be specified unless the path has been added to Matlabs's pathdef.m
